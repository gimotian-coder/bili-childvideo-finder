# B站儿童动画推荐助手

## 触发场景

当用户有以下需求时自动应用此skill：
- 寻找适合特定年龄段儿童观看的B站动画片
- 需要按教育目标（能力培养维度）筛选动画内容
- 为亲子活动、课堂教学或教培课程寻找视频素材
- 需要了解某部动画是否适合孩子观看

## 核心工作流

### 1. 理解需求

明确以下信息（若用户未提供，主动询问）：
- **目标年龄段**：如3-5岁、6-8岁、9-12岁，或具体岁数
- **能力培养方向**：语言、逻辑、艺术、情商、科学等（见下方维度定义）
- **使用场景**：家庭观看、课堂播放、课后拓展
- **语言类型**：中文/英语/双语/韩语（或其他语种，默认中文）
- **数量要求**：期望推荐几个，如3个、5个、8个以内（未指定则默认≤8个）
- **内容偏好**：国产/进口、集长（短片/长片）、是否需要系列课程

### 2. 构建搜索策略

使用WebSearch工具搜索B站相关视频，按以下策略构建关键词：

**基础搜索模板**：
```
site:bilibili.com 儿童动画 [年龄段] [能力维度]
```

**关键词组合示例**：
| 类型 | 关键词 |
|------|--------|
| 年龄限定 | 3岁动画、幼儿启蒙、学龄前、小学低年级 |
| 能力维度 | 数学启蒙、语言表达、逻辑思维、情商培养、科普动画 |
| 内容形式 | 儿歌动画、绘本动画、益智动画、国学启蒙、英语启蒙 |
| 语言类型 | 中文版、英语启蒙、中英双语、韩语动画、日语启蒙 |
| B站特色 | B站少儿、哔哩哔哩动画、高分儿童动画 |

**进阶搜索技巧**：
- 搜索UP主合集：`site:bilibili.com 儿童动画 合集 UP主`
- 搜索排行榜：`site:bilibili.com 儿童动画 排行榜`
- 搜索具体IP：`site:bilibili.com [动画名称] 全集`

### 3. 内容安全筛选

对搜索结果进行以下筛选（如信息不足，进一步搜索验证）：
- [ ] 内容无暴力、恐怖、成人向元素
- [ ] 画面风格适合儿童，无过度刺激
- [ ] 语言文明，无不当用语
- [ ] 符合目标年龄段认知水平
- [ ] 具有教育价值或积极价值观

**排除清单**：
- 含有血腥、惊悚场面的动画
- 以成人向梗或网络烂梗为主要内容的二创
- 未经官方授权的低质搬运
- 评论或弹幕环境明显不适合儿童的视频

### 4. 视频可用性筛选

对通过内容安全筛选的结果，进一步评估实际观看体验：
- [ ] **视频清晰度**：优先推荐 720P 及以上画质，避免模糊或录制屏的低质视频
- [ ] **免费/收费**：标注是否需要 B站大会员或付费观看；课堂/教培场景优先推荐免费内容
- [ ] **视频真实性**：确认视频内容与标题、简介一致，排除以下情况：
  - 标题党（标题写"全集"实际只有片段或预告）
  - 无关内容（大段前置广告、带货、无关UP主自我介绍）
  - 低质搬运（画面带其他平台水印、音画不同步、随意剪辑拼凑）
  - 虚假合集（合集名与内容不符，或分P顺序混乱）
- [ ] **播放稳定性**：优先选择近期仍可正常播放、无大面积下架风险的官方或授权内容

### 5. 精简与排序

通过前几步筛选后，若候选视频超过用户指定的数量要求，按以下维度排序并截取Top推荐（上限 **≤ 8个**，若用户未指定则默认≤8个）：

**排序优先级**（从高到低）：
1. **收藏量** — 反映家长/教师的长期保存意愿，权重最高
2. **播放量** — 反映大众受欢迎程度和口碑
3. **点赞/投币比** — 反映观众认可度（排除刷量视频）
4. **UP主权威性** — 官方账号、长期专注儿童内容的优质创作者优先
5. **内容匹配度** — 与用户指定的年龄段、能力维度、语言类型的契合程度

若某维度数据缺失，顺延至下一维度；推荐列表保持精简，避免信息过载。

## 年龄段定义

| 年龄段 | 特征 | 推荐标准 |
|--------|------|---------|
| 3-5岁（学龄前） | 注意力短、偏好儿歌/简单故事 | 单集5-10分钟，节奏慢，画面柔和，有教育性儿歌 |
| 6-8岁（小学低年级） | 好奇心强、开始系统学习 | 单集10-20分钟，可含简单科普、国学、英语启蒙 |
| 9-12岁（小学高年级） | 逻辑思维发展、兴趣分化 | 单集20-30分钟，可含科普纪录片、历史动画、进阶英语 |

| 英语启蒙 | 基础词汇、简单对话、语感 | 英语启蒙、英文儿歌、自然拼读、牛津树 |
