# B站儿童动画推荐助手

帮助家长、老师和亲子教培机构在 B站 快速找到适合不同年龄段儿童观看的动画片。

[English Version](README.md)

## 功能特性

- 年龄段分层推荐 — 支持 3-5岁（学龄前）、6-8岁（小学低年级）、9-12岁（小学高年级）
- 多语言支持 — 中文、英语、双语、韩语等多语种内容筛选
- 内容安全筛选 — 自动排除暴力、恐怖、成人向内容，保障儿童观看安全
- 视频可用性筛选 — 评估清晰度、免费/收费、视频真实性
- 智能排序 — 按收藏量、播放量、UP主权威性等多维度筛选 Top 推荐
- 数量可控 — 支持自定义推荐数量（默认不超过 8 个），避免信息过载
- 结构化输出 — 推荐列表包含 B站链接、适合年龄、推荐理由、使用建议等

## 适用场景

- 为孩子寻找适合年龄的动画片
- 为亲子活动、课堂教学或教培课程准备视频素材
- 了解某部动画是否适合孩子观看

## 使用方法

在 QoderWork 中直接输入你的需求，例如：

```
帮我找适合5岁孩子看的数学启蒙动画片，中文，推荐5个左右
```

```
我需要一些适合8岁孩子的英语启蒙动画，中英双语更好
```

```
为亲子教培课程找一些9-12岁孩子的科学科普动画
```

## 核心工作流

1. **理解需求** — 明确年龄段、语言类型、数量要求
2. **构建搜索策略** — 使用 B站搜索关键词组合精准定位内容
3. **内容安全筛选** — 排除不适合儿童观看的内容
4. **视频可用性筛选** — 评估清晰度、免费/收费、视频真实性
5. **精简与排序** — 按收藏量、播放量等维度筛选 Top 推荐（上限不超过 8 个）
6. **输出结构化推荐列表** — 生成包含链接、推荐理由、使用建议的完整推荐
